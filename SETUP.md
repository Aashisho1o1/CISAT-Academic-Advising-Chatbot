# CISAT Academic Advising Chatbot - Setup Instructions

## Prerequisites

- Node.js 18+ installed
- npm, pnpm, or yarn package manager

## Installation Steps

1. Create project folder:

```bash
mkdir cisat-advising-chatbot
cd cisat-advising-chatbot
```

2. Copy all files into the project folder.
3. Install dependencies:

```bash
npm install
# or
pnpm install
# or
yarn install
```

4. Run development server:

```bash
npm run dev
```

5. Open browser: Navigate to http://localhost:5173

## Project Structure

- `/src/app/` - Main application code
- `/src/app/screens/` - Screen components
- `/src/app/components/` - Reusable components
- `/src/styles/` - CSS and theme files
